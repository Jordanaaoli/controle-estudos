# 📚 Controle de Estudos

Aplicação full stack desenvolvida para gerenciar matérias e registrar sessões de estudo.

## 🚀 Tecnologias

* React
* Node.js
* Express
* PostgreSQL

## 💡 Funcionalidades

* Cadastro de matérias
* Listagem de matérias
* Registro de sessões de estudo
* Listagem de sessões com relacionamento entre matéria e tempo estudado

## 🧠 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar e demonstrar habilidades em desenvolvimento full stack, incluindo integração entre frontend, backend e banco de dados relacional.

## ⚙️ Como rodar o projeto

### Backend

```bash
cd backend
npm install
node index.js
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## 🔐 Variáveis de ambiente

Crie um arquivo `.env` dentro da pasta `backend` com:

```
DB_USER=postgres
DB_HOST=localhost
DB_NAME=controle_estudos
DB_PASSWORD=SUA_SENHA
DB_PORT=5432
```
