# Controle de Estudos

Projeto full stack para controle de matérias e sessões de estudo.

## Tecnologias utilizadas

- React
- Node.js
- Express
- PostgreSQL

## Funcionalidades

- Cadastro de matérias
- Listagem de matérias
- Registro de sessões de estudo
- Listagem de sessões com matéria relacionada
